شروع کار با DeepLander

این صفحه راهنمای شروع سریع پروژه DeepLander است. هدف این بخش این است که هر کاربر، حتی بدون تجربه قبلی، بتواند پروژه را روی سیستم خود اجرا و بررسی کند.

1️⃣ پیش‌نیازها

  • مرورگر مدرن: Chrome, Firefox, Safari یا Edge
  • اتصال به اینترنت برای بارگذاری منابع آنلاین
  • Git نصب شده روی سیستم (برای کلون کردن ریپازیتوری)
  • توصیه حرفه‌ای: نصب Visual Studio Code یا هر ویرایشگر متن پیشرفته

2️⃣ کلون کردن ریپازیتوری

git clone https://github.com/jumplander-readme/DeepLander.git
cd DeepLander

3️⃣ مشاهده پروژه

  1. فایل index.html را در مرورگر باز کنید
  2. یا از افزونه Live Server در VS Code استفاده کنید

4️⃣ نکات حرفه‌ای

  • برای هر تغییر، Branch جداگانه بسازید: 
    git checkout -b feature/your-feature-name
  • Commit با پیام کوتاه و واضح: 
    git commit -m "Add Persian introduction section for DeepLander"
  • Pull Request بسازید تا تغییرات بررسی و Merge شود

5️⃣ منابع آموزشی مرتبط

  • GitHub Docs – Getting Started
  • GitHub Pages Documentation
  • HTML Basics – MDN

About DeepLander

DeepLander یک وب‌سایت در حال توسعه است که با استفاده از هوش مصنوعی اختصاصی JumpLander ساخته می‌شود.

هدف پروژه ارائه یک Landing Page هوشمند و آموزشی برای کاربران فارسی‌زبان است.

اهداف اصلی پروژه

  • معرفی قابلیت‌ها و روش استفاده از هوش مصنوعی JumpLander
  • نمونه عملی Landing Page هوشمند و بهینه
  • آموزش GitHub و فرآیند مشارکت در پروژه

اهمیت برای کاربران فارسی‌زبان

  • پشتیبانی کامل از راست‌به‌چپ (RTL)
  • متن و لحن فارسی قابل فهم
  • نمونه واقعی از پروژه در حال توسعه
  • افزایش آگاهی از کاربردهای هوش مصنوعی

Roadmap پروژه DeepLander

  • MVP – معرفی اولیه: آماده‌سازی بخش معرفی و اولین نسخه HTML فارسی
  • نسخه 1.1: اضافه کردن آموزش HTML/CSS و نکات GitHub
  • نسخه 1.2: بهبود UX و ریسپانسیو
  • نسخه 2.0: افزودن تعاملات هوش مصنوعی و قابلیت‌های پیشرفته

ویژگی‌های در حال توسعه

  • متن و معرفی فارسی کامل
  • ساخت Pull Request نمونه و آموزشی
  • افزودن مثال‌های عملی آموزشی

ایده‌های آینده

  • ترجمه انگلیسی و دو زبانه شدن
  • افزودن ویدیو و تصویر برای آموزش بهتر
  • مستندسازی کامل برای مشارکت‌کنندگان
  • افزودن قابلیت‌های AI بیشتر و تجربه شخصی‌سازی شده

راهنمای مشارکت در پروژه DeepLander

  1. یک Issue مناسب انتخاب کنید
  2. Branch جدید بسازید: 
    git checkout -b feature/your-feature-name
  3. تغییرات خود را اعمال کنید
  4. Commit با پیام کوتاه: 
    git commit -m "Add Persian introduction section"
  5. Pull Request بسازید و توضیح دهید

قوانین حرفه‌ای

  • هر PR مرتبط با یک Issue باشد
  • Branch جدا برای هر تغییر ایجاد کنید
  • پیام Commit کوتاه و واضح باشد
  • از ساختار Markdown/HTML پروژه پیروی کنید

یادگیری GitHub با DeepLander

  • Issues: مدیریت وظایف و باگ‌ها
  • Pull Requests: ارائه تغییرات و بررسی کد
  • Branches: ساخت مسیر مستقل برای توسعه
  • Projects: مدیریت نقشه راه و وظایف
  • Wiki: مستندسازی و ارائه راهنما

مثال عملی

  1. Issue: "[Content] معرفی DeepLander"
  2. Branch: content/deeplander-introduction-fa
  3. Commit: 
    git commit -m "Add Persian introduction section"
  4. Pull Request: Merge به main پس از بررسی
