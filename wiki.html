شروع کار با DeepLander

این بخش به کاربران کمک می‌کند پروژه را به سرعت اجرا کنند.

پیش‌نیازها

  • مرورگر مدرن: Chrome, Firefox, Safari یا Edge
  • اتصال به اینترنت برای بارگذاری منابع آنلاین
  • Git نصب شده روی سیستم
  • توصیه: Visual Studio Code یا هر ویرایشگر متن پیشرفته

کلون کردن ریپازیتوری

git clone https://github.com/jumplander-readme/DeepLander.git
cd DeepLander

مشاهده پروژه

  1. باز کردن index.html در مرورگر
  2. یا استفاده از Live Server در VS Code

درباره DeepLander

DeepLander یک وب‌سایت هوشمند ساخته شده با هوش مصنوعی JumpLander است.

اهداف اصلی

  • معرفی قابلیت‌های JumpLander
  • نمونه عملی Landing Page هوشمند
  • آموزش GitHub و مشارکت در پروژه

نقشه راه پروژه

  • MVP: نسخه اولیه HTML فارسی
  • نسخه 1.1: آموزش HTML/CSS و GitHub
  • نسخه 1.2: بهبود UX و ریسپانسیو
  • نسخه 2.0: افزودن تعاملات هوش مصنوعی

راهنمای مشارکت

  1. یک Issue انتخاب کنید
  2. Branch جدید بسازید: 
    git checkout -b feature/your-feature-name
  3. تغییرات را اعمال کنید
  4. Commit با پیام کوتاه: 
    git commit -m "Add Persian introduction section"
  5. Pull Request بسازید

یادگیری GitHub با DeepLander

  • Issues: مدیریت وظایف و باگ‌ها
  • Pull Requests: ارائه تغییرات و بررسی کد
  • Branches: مسیر مستقل برای توسعه
  • Wiki: مستندسازی و راهنما
